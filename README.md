# Omni Marketing Website

Static multi-page website for Omni Marketing.

## Pages

- `index.html`
- `about.html`
- `services.html`
- `work.html`
- `contact.html`

## Notes

- The site is dependency-free and can be opened directly in a browser.
- The contact form prepares a `mailto:` inquiry for `hello@omnimarketingonline.com`.
- The visual direction uses the supplied Omni Marketing logo as reference and includes a self-contained SVG mark in `assets/omni-logo.svg`.
